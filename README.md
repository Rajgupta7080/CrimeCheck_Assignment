# CrimeCheck_Assignment

It is a coding assignment given by crimecheck, Where i have to make a notice-board full-stack app.

## Backend Api :- https://crimecheck-notice-board.onrender.com

<li> Backend deployed on render, All api's are working fine.</li>

## Frontend Deployed Link :- https://public-notice-board.netlify.app/

<li>Frontend deployed on netlify</li>

# Functionality

<li>User can login by username</li>
<li>Username must be Alphanumeric</li>
<li>User can publish notice</li>
<li>User can see all public notes also</li>

# Tech Stacks

## Backend

<li>Node.js</li>
<li>express.js</li>
<li>MongoDb</li>

## Frontend

<li>React.js</li>
<li>Css</li>
